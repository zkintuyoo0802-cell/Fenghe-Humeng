离线使用与分享

  • 推荐转发（单文件）：运行 python scripts/build-standalone.py 生成 index.standalone.html，只发该文件即可；内联样式与数据，双击即可用，不依赖子文件夹。
  • 文件夹版：将整个 fenghe-dashboard 打 ZIP（须包含 css/、js/、data/），解压后双击 index.html。勿只复制单个 HTML。
  • 可选在线托管：将该文件夹上传到 GitHub Pages、Cloudflare Pages、Netlify 等任意静态托管，即可通过网址访问（HTTP 模式下需 data/keywords.json 与 data/monthly-insights.json）。

数据：scripts/embed-data.py 合并 JSON → js/data.js。单文件版：embed-data.py 后运行 python scripts/build-standalone.py。离线双击 index.html 时浏览器不允许用 fetch 读本地 JSON，故必须存在已嵌入数据的 js/data.js 或使用 index.standalone.html。
